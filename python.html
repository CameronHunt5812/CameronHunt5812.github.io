The Word Search Solver

Description

My program solves a word searches. It uses loops to compair each letter in the word search to the firs letter of the word it is looking for. When it finds a match it checks the adjacent letters, if it finds a letter that matches the second letter of the word, it will follow the angle set by those two letters. If it dose not find the word it returns to looking at the ajacent letters and comparing them to words that start with the same letter. Befor the program can solve a wordsearch you need to put it into a text file with the width the highth and the number of words in the word search above it, then the word search with no spaces, and after it, each on its own line, the words to look for.
